# Pytest Learning Repository

This repository is dedicated to my progress and exercises in the Udemy course: **"Pytest Course: Practical Testing of Real-World Python Code"**.

## Project Overview
The goal of this repository is to master the `pytest` framework through hands-on practice. It contains various tests and small applications that demonstrate testing principles, from basic data structures to complex business logic.

### Repository Content
- **`tests/`**: Basic unit tests for Python core features (lists, dicts, math, strings).
- **`introduction/`**: Getting started with workers and basic testing patterns.
- **`practice-orders-processing/`**: A more complex example of testing an e-commerce order processing system.

## About the Course
This course covers how to test real-world Python applications with `pytest`, covering everything from simple unit tests to full CI/CD automation.

**Key learnings:**
- Using `pytest` to make code maintainable and robust.
- Applying testing to legacy systems and new projects (TDD).
- Integrating tests into CI/CD pipelines.
- Achieving high test coverage without over-engineering.
